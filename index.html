Arduino Skittle Sorting Machine


Technologies

  • Arduino Relay Motor Shield

  • Arduino Uno Micro Controller

  • 1 Servo Motor

  • Stepper Motor

  • 3D printer (OpenScad)

  • Laser Cutter (Adobe Illustrator)

  • 1 Servo Motor

  • Bag of Skittles

  • OpenScad 3D modeling sofware

  • Adobe Illustrator 2.5D design

Description

The Skittle Sorting is a machine with the ability to sort skittles based on their color. Skill used to create our machine included being able to design in 3D printed object using Openscad, design 2.5D laser cut prototypes using adobe illustrator and the Arduino language to write the software controlling each mechanism within the skittle which include a skittle feeder, sorting mechanism, and a different color distribution slide.
